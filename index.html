Instruções

Press [p] para pausar, ou [ESC] para abandonar o jogo.

Vira é um jogo de memória com cartas cronometradas. Clique nas cartas verdes para ver qual símbolo elas descobrem e tente encontrar o símbolo correspondente abaixo das outras cartas.

Descubra dois símbolos correspondentes de uma só vez para eliminá-los do jogo.

Elimine todas as cartas o mais rápido possível para ganhar o jogo. Divirta-se VIRANDO!
